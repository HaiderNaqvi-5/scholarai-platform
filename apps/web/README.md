# ScholarAI Web App

This directory is the new Next.js web surface for ScholarAI's MVP.

Current scope:

- establish the app-router shell
- align base styling tokens with `docs/scholarai/03_brand_and_design_system.md`
- reserve room for student-facing MVP routes without implementing features yet
- provide placeholder routes for onboarding, profile, recommendations, dashboard, document feedback, and interview practice

Out of scope in this scaffold step:

- product features
- API integration
- recommendation UI
- admin curation screens
- generic dashboard-template styling
