# ScholarAI — Skills Matrix & Knowledge Requirements

---

## Required Skills by Role

### AI/ML Engineer
| Skill | Level | Used In |
|---|---|---|
| Python (3.11+) | Advanced | All ML code |
| scikit-learn | Advanced | Feature engineering, model training |
| XGBoost | Advanced | Recommendation engine |
| SHAP / LIME | Advanced | Explainability module |
| LangChain | Intermediate | LLM orchestration, RAG pipeline |
| Whisper (OpenAI) | Intermediate | Interview simulator |
| MLflow | Intermediate | Experiment tracking |
| Optuna | Intermediate | Hyperparameter tuning |
| Sentence-BERT | Intermediate | Text embeddings |
| Pandas / NumPy | Advanced | Data manipulation |

### Backend Engineer
| Skill | Level | Used In |
|---|---|---|
| Python (3.11+) | Advanced | All backend code |
| FastAPI | Advanced | REST API server |
| SQLAlchemy 2.0 | Advanced | ORM, database models |
| Alembic | Intermediate | Database migrations |
| PostgreSQL | Advanced | Primary database |
| Redis | Intermediate | Caching, task queue broker |
| Celery | Intermediate | Async task processing |
| Docker | Intermediate | Containerization |
| JWT / OAuth2 | Intermediate | Authentication |
| Pydantic | Advanced | Data validation |

### Data Engineer
| Skill | Level | Used In |
|---|---|---|
| Playwright | Advanced | Web scraping |
| Neo4j / Cypher | Advanced | Knowledge graph |
| Python | Advanced | Scraper scripts, data pipeline |
| LLM APIs | Intermediate | Gemini structured extraction |
| Data modeling | Advanced | Schema design, ETL |
| Redis | Intermediate | Deduplication cache |

### Frontend Engineer
| Skill | Level | Used In |
|---|---|---|
| Next.js 16 | Advanced | Frontend application |

| CSS / Tailwind | Advanced | UI styling |
